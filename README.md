# AWS IoT SiteWise

여기서는 IoT SiteWise에 대해 설명합니다. 

AWS IoT SiteWise는 industrial 장비에서 수집된 데이터를 쉽게 모으고, 저장하고, 정리하여 모니터링하는 managed service입니다. 트래픽이 늘어날 경우에 자동으로 scale up하고 유지보수등의 수고를 AWS에서 해주므로 비지니스 로직 개발에 집중할 수 있습니다.
